A Touristic Guide to World Monuments

About the Developer

My name is Hani Mohtar, and I am a second-year Computer Science student at the Lebanese University, Faculty of Sciences. This project represents my work for the Web Development course.

Project Purpose

The "Touristic Guide to World Monuments" was created to provide travelers with an accessible, responsive digital guide to historical sites. It focuses on clean navigation and informative content to enhance the user's understanding of global cultural heritage.

Tools & Technologies

  • HTML5: For semantic web structure.
  • CSS3: Used Flexbox, Grid and Media Queries for responsive design.
  • JavaScript: Implemented for form validation and UI interactions.
  • Google Maps API: Integrated for location services.
